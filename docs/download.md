## Download
The latest release of LibreCrypt, together with its source code, may be downloaded from the GitHub project at: [https://github.com/t-d-k/librecrypt](https://github.com/t-d-k/librecrypt)

PGP, MD5 and SHA-1 signatures for this software may also be found on the above web site.

If you would like to build your own copy of LibreCrypt, you will also need the SDeanUtilsXE package (required for building the GUI), which may be downloaded from: [GitHub](https://github.com/t-d-k/librecrypt)
